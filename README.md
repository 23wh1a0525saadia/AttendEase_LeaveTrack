# Attendease – LeaveTrack Web Application

Attendease is a web-based leave management system designed to simplify the leave request and approval process for students and administrators.

## Features
- Student authentication and leave request submission
- Admin dashboard for approving or rejecting leave requests
- Role-based access for students and admins
- File upload support for leave documents
- Clean UI with form validation

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MySQL

## Project Structure
- `app.py` – Main Flask application
- `templates/` – HTML templates
- `static/` – CSS and JavaScript files
- `uploads/` – Uploaded documents

## How to Run
1. Clone the repository
2. Install dependencies from `requirements.txt`
3. Configure database credentials in `db_config.py`
4. Run `python app.py`
5. Open the application in your browser
